Task SQL
How many people are in the database? SELECT COUNT( id ) FROM users
How many people are in New York? SELECT COUNT(id) FROM `users` Where state = 'NY';
How many people have opted out in total? SELECT SUM(opt_out) FROM users; SELECT COUNT(opt_out) FROM users WHERE opt_out > 0;
Show each state and the number of people in that state SELECT `state`, COUNT(id) FROM users GROUP BY state;
How many total cars are owned? SELECT SUM(cars) FROM users;
How many total cars are owned by people in FL? SELECT SUM(cars) FROM users WHERE state = 'FL'
What are the most number of cars owned by any one person? SELECT MAX(cars) FROM users;
What are the least number of cars owned by any one person in NY? SELECT MIN(cars) FROM users WHERE state = "NY"
How many cars are owned by each family (identified by last name)? SELECT lname, SUM(cars) FROM users GROUP BY lname ORDER BY SUM(cars);
How many cars are owned in each state by Smiths? SELECT state COUNT(cars) FROM users WHERE lname = Smith GROUP BY state;
Show the states where Smiths own more than 2 cars SELECT state FROM users WHERE lname = 'Smith' GROUP BY state Having SUM(cars) > 2;
